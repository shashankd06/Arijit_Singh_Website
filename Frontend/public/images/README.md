# Images Directory

## Background Image

Please place your Arijit Singh concert performance image in this directory with the filename:

**`arijit-concert.jpg`**

### Image Specifications

- **Filename:** `arijit-concert.jpg` (or `.png`, `.jpeg`)
- **Recommended Size:** 1920x1080 pixels or larger
- **Format:** JPG, PNG, or JPEG
- **Location:** `frontend/public/images/arijit-concert.jpg`

### Image Description

The image features Arijit Singh performing live on stage with an acoustic guitar, bathed in warm red/orange/purple stage lights with a dramatic, smoky atmosphere. The image captures the energy and emotion of a live concert performance.

### Alternative Filenames

If your image has a different name, you can:
1. Rename it to `arijit-concert.jpg`, or
2. Update the inline styles in `frontend/src/pages/Home.js` to match your filename (line 18)

The image will be used as the background for the home page hero section.
